نظرة عامة
تطبيق ويب متكامل للاستماع إلى تلاوات القرآن الكريم بصوت ثلاثة من أشهر القراء:

مشاري العفاسي

عبدالرحمن السديس

ورش عن نافع

تم تصميم التطبيق بواجهة عربية حديثة وسهلة الاستخدام، مع إمكانية البحث، وإضافة السور إلى المفضلة، والتحكم الكامل في التشغيل.

متطلبات التشغيل
أي متصفح ويب حديث (Chrome, Firefox, Safari, Edge)

اتصال بالإنترنت لتحميل التلاوات

محرر أكواد (VS Code، Sublime Text، أو أي محرر نصي) لتعديل الكود

طريقة التشغيل
الطريقة الأولى: تشغيل مباشر
احفظ الملف باسم quran-player.html

افتح الملف مباشرة في المتصفح (Double-click أو السحب إلى المتصفح)

الطريقة الثانية: عبر محرر الأكواد
افتح محرر الأكواد المفضل لديك

أنشئ ملفًا جديدًا باسم index.html

انسخ الكود بالكامل والصقه في الملف

احفظ الملف

استخدم خاصية "Live Server" في VS Code (إذا كانت متوفرة) أو افتح الملف مباشرة في المتصفح

في VS Code:
قم بتثبيت إضافة Live Server

انقر بزر الماوس الأيمن على الملف

اختر Open with Live Server

الميزات الرئيسية
🎵 المشغل الصوتي
تشغيل وإيقاف التلاوة

الانتقال بين السور (التالي/السابق)

شريط تقدم تفاعلي للنقر السريع

عرض الوقت الحالي والمدة الإجمالية

تكرار السورة تلقائيًا

🔊 التحكم بالصوت
شريط تحكم بمستوى الصوت

أزرار رفع/خفض الصوت

حفظ إعدادات الصوت تلقائيًا

📚 قائمة السور
عرض جميع السور الـ 114 مرتبة

البحث عن سورة بالاسم أو الرقم

تصميم شبكي متجاوب مع جميع الأجهزة

⭐ المفضلة
إضافة السور إلى قائمة المفضلة

إزالة السور من المفضلة

حفظ المفضلة في المتصفح (Local Storage)

صفحة منفصلة لعرض السور المفضلة

🎤 تبديل القراء
تبديل سهل بين القراء الثلاثة

تحديث روابط التلاوة تلقائيًا

الحفاظ على حالة التشغيل الحالية

🎮 اختصارات لوحة المفاتيح
Space: تشغيل/إيقاف

سهم اليمين →: السورة التالية

سهم اليسار ←: السورة السابقة

R: تفعيل/إلغاء تكرار السورة

سهم لأعلى ↑: رفع الصوت

سهم لأسفل ↓: خفض الصوت
